# albaayo-android
알바 관리 앱 - 안드로이드(JAVA)

```
사용기술: Android(JAVA), retrofit2, Stomp 프로토콜, 카카오 주소 검색 API, 카카오 지도 API
```
## Do
- retrofit2를 이용하여 백엔드 서버와 API 통신

- Stomp 프로토콜을 이용하여 채팅 기능 구현

- 카카오 지도 API를 이용하여 현재 위치와 가게 위치 거리 체크
